<h2 align="center">Использование</h2>

Использование с раздачей статики:

```bash
npm run start
```

<h2 align="center">Разработчикам</h2>

Установка зависимостей:
```bash
npm install
```

Запуск с обновлением для разработчиков:
```bash
npm run dev
```

сборка стабильной версии:
```bash
npm run build
```
